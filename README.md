## [MakeyBot Workshop](https://github.com/jimTheSTEAMClown/MakeyBot-Workshop)
3D Printed, Arm, Tentacle, Tail waving Flat Robot, where you can learn some 3D Printing skills, some Wiring and Soldering skills, and some Coding sills, in both Arduino (C++) and Python (Raspberry Pi of other Python Hardware)
